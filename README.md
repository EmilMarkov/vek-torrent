# VEK Torrent

**Удобный торрент-менеджер с поиском по rutracker в тёмной теме.**

Быстрый поиск, мгновенные фильтры, красивые страницы раздач и встроенное
управление загрузками поверх qBittorrent — в одном настольном приложении.

## Возможности

- **Умный поиск по rutracker.** Живой поиск (lazy-search) с задержкой ввода и
  отменой устаревших запросов — результаты появляются по мере набора текста.
- **Мгновенная фильтрация.** Уточняйте уже загруженную выдачу без повторных
  запросов к трекеру: по словам (в том числе исключающим через `-слово`),
  размеру, числу сидов, категориям и статусу проверки. Гибкая сортировка на
  лету — доработка того, чего не хватает в самом rutracker.
- **Красивые страницы раздач.** Содержимое берётся с rutracker, но отображается
  собственным аккуратным интерфейсом: спойлеры, цитаты, код, списки,
  изображения и форматирование — без нагромождения оригинальной вёрстки.
- **Встроенный менеджер загрузок.** Управление закачками на базе qBittorrent:
  добавление в один клик, пауза/возобновление, удаление, живой прогресс,
  скорости и статистика в реальном времени.
- **Внешний REST API.** Используйте функциональность приложения в своих
  программах и скриптах: поиск, страницы раздач и управление загрузками через
  HTTP с документацией OpenAPI (Swagger UI).
- **Тёмная тема.** Продуманный, спокойный интерфейс, комфортный для глаз.

Готовые сборки доступны на странице
[**Releases**](https://github.com/OWNER/vek-torrent/releases):

> [**qBittorrent**](https://www.qbittorrent.org/download). Приложение само
> запускает его в фоне — отдельная настройка qBittorrent не требуется. Если он
> установлен в нестандартную папку, укажите путь в настройках.

## С чего начать

1. Откройте **Настройки** и введите логин и пароль от rutracker, затем нажмите
   **Сохранить и войти**. Если трекер запросит капчу — появится окно с картинкой
   для ввода кода.
2. Перейдите на вкладку **Поиск** и введите запрос. Уточняйте результаты
   фильтрами в правом верхнем углу.
3. Нажмите на раздачу, чтобы открыть её страницу, и кнопку **Скачать** — закачка
   появится на вкладке **Загрузки**.

## Внешний API

Приложение может поднимать локальный REST API для интеграций.

1. В **Настройках → Внешний API** включите API и при необходимости измените
   адрес и порт (по умолчанию `127.0.0.1:8737`).
2. Скопируйте **токен доступа** и нажмите **Применить настройки API**.
3. Обращайтесь к API, передавая токен в заголовке `Authorization`:

```bash
# Проверка статуса
curl -H "Authorization: Bearer ВАШ_ТОКЕН" http://127.0.0.1:8737/api/v1/status

# Поиск
curl -X POST http://127.0.0.1:8737/api/v1/search \
  -H "Authorization: Bearer ВАШ_ТОКЕН" \
  -H "Content-Type: application/json" \
  -d '{"query": "linux mint"}'

# Добавить раздачу в загрузки по идентификатору темы
curl -X POST http://127.0.0.1:8737/api/v1/downloads/topic/6335144 \
  -H "Authorization: Bearer ВАШ_ТОКЕН" \
  -H "Content-Type: application/json" -d '{}'
```

Интерактивная документация (Swagger UI) доступна в браузере по адресу
`http://127.0.0.1:8737/docs`, спецификация OpenAPI — `/api-docs/openapi.json`.
Эндпоинт `/api/v1/health` доступен без токена для проверки доступности.

## Технологии

Настольное приложение на **Tauri 2** (Rust) с интерфейсом на **React 19** и
**TypeScript**. Поиск и разбор страниц — на Rust; загрузки — через
**qBittorrent Web API**; внешний API — на **axum** с OpenAPI.

## Сборка из исходников

Потребуются [Rust](https://rustup.rs) (stable), [Node.js](https://nodejs.org)
22+ и [системные зависимости Tauri](https://tauri.app/start/prerequisites/)
для вашей ОС.

```bash
# Установка зависимостей фронтенда
npm install

# Запуск в режиме разработки (горячая перезагрузка)
npm run tauri dev

# Сборка установщика под текущую платформу
npm run tauri build
```

Полезные команды при разработке:

```bash
npm test                 # тесты фронтенда (Vitest)
npm run lint             # ESLint
cargo test --workspace   # тесты Rust
cargo clippy --workspace --all-targets
```

